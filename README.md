# TEXT PREPROCESSING
This project converts complex unformatted text to simple formatted texts.

## Getting Started
These instructions will get you a copy of the project up and running on your local machine.

### Prerequisites
You should first install all the dependency libraries by running the following command
```
pip install -r requirements.txt
```

## Contents And Explainantion of How To Run The Code
Execute.py takes the name of the excel file containing the raw reviews under 'reviews' column
modules.py contains fuctions of each functional modules of text preprocessing
The output will be in the TextPreprocesser.xlsx
You can change the output filename in Execute.py
Once you have added the input and output filenames in the Execute.py file run the Execute.py
```
python Execute.py
```
